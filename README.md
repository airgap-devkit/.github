# airgap-devkit

Air-gapped developer toolkit — install, manage, and distribute developer tools
in network-restricted environments. Runs entirely offline. Zero CDN dependencies.

| Repo | Purpose |
|---|---|
| [devkit](../devkit) | Engine + pip package + web UI |
| [tools](../tools) | Default tool manifests and install scripts |
| [teams](../teams) | Forkable template for team tool images |
| [prebuilt](../prebuilt) | Optional prebuilt binary archives |

## Quick Start
pip install airgap-devkit
airgap-devkit
